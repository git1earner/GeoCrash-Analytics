# GeoCrash Analytics

A geospatial analytics and machine learning project exploring traffic accident patterns across the United States.

## Objectives

- Identify accident hotspots
- Analyze temporal traffic patterns
- Explore weather-related accident trends
- Visualize accident distributions using interactive maps
- Apply machine learning techniques for clustering and risk analysis

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Folium
- Plotly
- Jupyter Notebook

## Dataset

US Accidents Dataset:
https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents

## Project Status

Currently under development.

- Interactive accident hotspot maps
- Temporal trend analysis
- Severity analysis
